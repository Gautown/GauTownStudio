# Decap CMS标签使用指南

本文档详细说明了如何在Decap CMS中为内容添加和管理多个标签。

## 标签功能概述

标签功能允许您为内容添加多个分类标识，便于内容组织和检索。每个内容可以拥有零个、一个或多个标签。

## 在Decap CMS中添加标签

### 1. 打开内容编辑页面

1. 登录到Decap CMS管理界面
2. 选择您要编辑的内容集合（如"作品"、"工具"等）
3. 点击要编辑的内容项，或创建新内容

### 2. 添加标签

在内容编辑页面中：

1. 找到"标签"字段（通常在表单的中部位置）
2. 在输入框中直接输入标签，多个标签之间用逗号分隔
3. 例如：`技术, 教程, 前端, React, JavaScript`
4. 标签会自动保存为逗号分隔的字符串

### 3. 编辑标签

1. 要修改标签：直接在输入框中编辑文本
2. 可以添加新标签或删除现有标签
3. 确保标签之间用逗号分隔

### 4. 保存内容

完成标签添加后，点击页面顶部的"保存"按钮保存您的更改。

## 标签最佳实践

### 1. 标签输入格式

- 使用逗号（,）分隔多个标签
- 标签之间可以有空格，系统会自动去除多余空格
- 例如：`技术, 教程, 前端` 或 `技术,教程,前端` 都是有效的

### 2. 标签命名规范

- 使用简洁、明确的词语作为标签
- 避免使用特殊字符
- 保持标签风格一致（全部使用中文或全部使用英文）
- 避免创建过于具体的标签，优先使用通用标签

### 3. 标签数量建议

- 每篇文章建议使用3-8个标签
- 避免使用过多标签，以免降低分类效果
- 确保每个标签都有实际意义

### 4. 常用标签示例

根据不同内容类型，可以使用以下类型的标签：

#### 作品集标签
- 项目类型：网站、应用、设计等
- 技术栈：React、Vue、Angular等
- 行业领域：电商、教育、金融等

#### 工具标签
- 工具类型：开发工具、设计工具、效率工具等
- 平台：Windows、Mac、Web等
- 功能：自动化、协作、分析等

#### 笔记标签
- 主题：技术、设计、管理等
- 难度级别：初级、中级、高级等
- 学习类型：教程、概念、实践等

## 标签在前端的使用

标签数据会以字符串形式存储在内容的frontmatter中，但在Astro中会自动转换为数组：

```yaml
---
title: 示例项目
tags: 技术, 教程, 前端
---
```

在前端代码中，您可以通过以下方式访问标签：

```javascript
// 在Astro组件中
---
const { tags } = Astro.props; // tags现在是一个数组
---

{tags && tags.length > 0 && (
  <div class="tags">
    <h3>标签：</h3>
    <ul>
      {tags.map(tag => (
        <li class="tag">{tag}</li>
      ))}
    </ul>
  </div>
)}
```

### 标签显示位置

标签会在以下位置显示：

1. **内容详情页面**：在文章标题下方显示该内容的所有标签
2. **侧边栏热门标签**：显示整个网站中使用频率最高的标签

### 标签样式

标签使用以下CSS类进行样式设置：

- `.portfolio-tags`：标签容器
- `.tag-list`：标签列表容器
- `.tag`：单个标签样式

## 常见问题解答

### 1. 为什么我添加的标签没有显示？

- 确保已正确保存内容
- 检查标签是否用逗号正确分隔
- 确认标签字段是否正确配置在config.yml中
- 检查内容详情页面模板是否包含标签显示代码
- 确认全局CSS中是否包含标签样式

### 2. 可以使用中文标签吗？

可以，Decap CMS完全支持中文标签。建议保持标签语言风格的一致性。

### 3. 标签有数量限制吗？

技术上没有硬性限制，但建议每篇文章使用3-8个标签以保持良好的分类效果。

### 4. 如何删除所有标签？

清空标签输入框中的所有内容即可。

### 5. 为什么侧边栏的热门标签没有更新？

- 确保内容已正确保存并部署
- 检查页面模板是否正确收集和处理标签数据
- 确认标签数据是否传递给Sidebar组件

## 高级用法

### 1. 标签搜索功能

可以通过标签实现内容搜索功能，在前端代码中过滤具有特定标签的内容。

### 2. 标签云

可以基于所有内容的标签生成标签云，显示标签的使用频率。

### 3. 相关内容推荐

通过分析内容的共同标签，可以实现相关内容推荐功能。
